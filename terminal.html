Linux Terminal  —  Find the Flag!

A beginner-friendly sandbox. Explore the file system and capture the hidden flag.

Goal: Find a file containing a secret flag hidden somewhere in the directory tree.
Tip: Start with ls to list files, cd <folder> to enter a folder, cat <file> to read a file.
guest@ctf-box:~
guest@ctf-box:~$
ls
List files in current folder
ls -la
List all files with details
pwd
Show current directory path
cd <folder>
Enter a folder
cd ..
Go back to parent folder
cat <file>
Print file contents
echo <text>
Print text to screen
file <name>
Describe a file or folder
whoami
Show current username
uname -a
System information
history
Show command history
clear
Clear the terminal screen
help
Show all available commands
