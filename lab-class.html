📘 What is a Class?

Calling Methods

A class is like a blueprint or recipe. It tells JavaScript how to create objects. Think: • Class = Blueprint • Object = Real thing made from blueprint Example: A Car blueprint can create many cars. The constructor gives starting values when the object is created. Simple idea: A class keeps data + actions together.

// Blueprint (Class Definition) class Animal { constructor(name) { this.name = name; } speak() { return `${this.name} makes a sound.`; } } // Creating an instance const dog = new Animal("Rex"); dog.speak(); // Output: Rex makes a sound.

Every class has 3 main parts: ① Class Name → Blueprint name ② Constructor → Special setup function that runs automatically when you create an object ③ Methods → Actions your object can do Example: new MathOps(10,5) This sends: a = 10 b = 5 Constructor stores them: this.a = a this.b = b Easy Memory Trick: constructor = setup methods = actions

Constructor is a special function that runs automatically when you create an object using new. It sets up the object’s starting values, like storing a and b inside this.a and this.b.

class MathOps { // constructor runs automatically constructor(a, b) { // Save values in object this.a = a; this.b = b; } add() { return this.a + this.b; } subtract() { return this.a - this.b; } } const calc = new MathOps(10, 5); calc.add(); // 15 calc.subtract(); // 5

To use a class: Step 1: Define class Step 2: Create object using new Step 3: Call methods with dot notation Example: calc.sum() Meaning: “Run the sum method on calc”

// 1. Define class Calculator { constructor(x, y) { this.x = x; this.y = y; } sum() { return this.x + this.y; } } // 2. Instantiate const calc = new Calculator(10, 5); // 3. Call calc.sum(); // Output: 15
object.method() → tells object what action to perform

// MathOps.js ← class definition class MathOps { constructor(a, b) { this.a = a; this.b = b; } add() { return this.a + this.b; } subtract() { return this.a - this.b; } multiply() { return this.a * this.b; } divide() { if (this.b === 0) { return "Cannot divide by zero"; } return this.a / this.b; } }
// main.js ← using the class // Create object const m = new MathOps(10, 5); // Call methods m.add(); // 15 m.subtract(); // 5 m.multiply(); // 50 m.divide(); // 2 // Reuse with new values const m2 = new MathOps(7, 3); m2.multiply(); // 21
